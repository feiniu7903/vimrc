
## vim environment

### how to do ?

```shell
git clone https://github.com/solome/vimrc.git ~/.vim
cd ~/.vim/bundle
git clone https://github.com/kien/ctrlp.vim ctrlp
git clone https://github.com/scrooloose/nerdtree
git clone https://github.com/tpope/vim-fugitive
cd ~/.vim
python setup.py
```
